# Adding Content from Password-Protected Pages

This guide helps you transfer content from your original password-protected portfolio pages to the new HTML site.

## Option 1: Manual Content Extraction

For each project page on your original site:

### Step 1: Access the Page

1. Go to https://www.nevenarmanios.com/
2. Click on a portfolio item
3. Enter the password: `watermellon`
4. The project content will display

### Step 2: Copy the Content

1. Select and copy all text content from the page
2. Note the structure (headings, paragraphs, lists)
3. Save images by right-clicking and "Save image as..."
4. Note image placement and captions

### Step 3: Fill in the Template

1. Open the corresponding project file in `projects/` folder
2. Find the template file (e.g., `segment-opportunity-simulator.html`)
3. Replace placeholder text with your actual content
4. Update image paths to point to your saved images

### Content to Extract

For each project, capture:

- **Project Title**: The main heading
- **Metadata**: Role, project type, year/timeline
- **Overview/Introduction**: Opening paragraph(s)
- **Challenge/Problem**: What problem you were solving
- **Process/Approach**: Your design methodology
- **Solution**: What you created
- **Results/Impact**: Outcomes, metrics, feedback
- **Key Learnings**: Insights gained (if present)
- **Images**: All project visuals with their captions

## Tips for Content Transfer

### Maintain Consistency

- Use similar section structure across all projects
- Keep heading hierarchy consistent (h3 for section titles)
- Maintain similar paragraph lengths

### Optimize for Web

- Break long paragraphs into shorter ones (3-5 sentences)
- Use descriptive headings that can be skimmed
- Place images strategically to break up text

### Image Best Practices

- Save images at web-optimized sizes (max 1920px width)
- Use descriptive file names (e.g., `persona-pathways-hero.jpg`)
- Compress images without losing quality (use tools like TinyPNG)
- Always include alt text for accessibility

### Don't Forget

- Project metadata (role, type, year)
- Image captions where appropriate
- Links to live projects or demos (if applicable)
- Navigation between projects (update "Next Project" links)
